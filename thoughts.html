On convexity

The convex combination of points \(x_1\) and \(x_2\) is, by definition, \(x(\theta)=\theta x_2 + (1-\theta) x_1\). Now, suppose \(x\) represents a production design For example, the components may represent lengths and powers of an electronic board. . Then, \(x\) can be interpreted as the mixture of the charachteristics of \(x_1\) and \(x_2\). But if we rewrite this in a centered fashion, namely \(x(\theta)=x_1 + \theta (x_2-x_1)\), \(x\) can now be seen as a base point \(x_1\) (the original design) being adapted to \(x_2\): conceptually, we have established a preference between the two! Mathematically, this duality is useful in proofs to reduce dimensionality: see page 33 here.

Least-squares linear regression has a physical interpretation that is just lovely! Given a linear model for the data (a line), attach to each data point a spring and connect the other end to the line (orthogonally). Since elastic energy is proportional to the square of deviation, the only mechanically stable model is that of Least-squares. See this post for some awesome visualizations.

We can employ \(\mathcal{l}_1\)-norm approximation for outlier detection, provided the model is linear. Just run the regression, and compute the residuals: this will be both computationally cheap (it's an LP) and allow outliers (\(\mathcal{l}_1\) is the best convex choice for that). The largest residuals will indicate outliers.
